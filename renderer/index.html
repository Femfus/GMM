About this project

I began developing the GTA mod manager after noticing that many of my friends struggled with the manual modding process. To make things easier, I created this project as a simple and efficient solution for both myself and them, allowing everyone to enjoy GTA V without needing to learn the complexities of mod installation.

This project will always be free and if you bought this program you were scammed.
